# Smart Study Planner

## Live Demo

Frontend: https://smart-study-planner-tl82.onrender.com


## Project Overview

Smart Study Planner helps students manage subjects, track study plans, and organize learning.

## Features

* User Signup
* User Login
* Forgot Password
* Dashboard
* Subject Management
* Study Planner
* dark mode
* Mobile Responsive

## Tech Stack

Frontend:

* React.js
* CSS

Backend:

* FastAPI
* Python

Database:

* MySQL

Deployment:

* Render

## How To Run

Frontend:
npm install
npm run dev

Backend:
pip install -r requirements.txt
uvicorn main:app --reload
